SIEFF 지속가능성 분석 리포트

투명한 데이터와 과학적 방법론으로 검증된 탄소발자국 분석

GHG Protocol 기준 측정

국제 표준인 GHG Protocol을 준수하여 Scope 1, 2, 3로 구분한 정확한 탄소배출량 측정. 전체 배출량의 93%가 Scope 3(간접배출)에 해당합니다.

✓ 국제 표준 준수

하이브리드 모델 혁신

온라인 참여율 96%를 달성하여 대규모 영화제임에도 불구하고 효율적인 탄소발자국을 실현. 온라인 참가자 1인당 7.75gCO₂로 매우 낮은 배출량을 기록했습니다.

✓ 96% 온라인 참여율

그린풋 탄소발자국 계산기

SIEFF가 자체 개발한 탄소발자국 계산기 '그린풋'을 통해 265명의 참가자가 자발적으로 이동 배출량 데이터를 제공. 실제 참가자 행동 패턴을 기반으로 한 정확한 측정을 실현했습니다.

✓ 265명 자발적 참여
